## Getting started

Make sure you have **NodeJS version 8** (or above) installed, then type the following commands:

```
git clone https://github.com/Enterprise-JS/koa-typescript-2018.git koa-typescript
cd koa-typescript
npm i
npm run start-watch
```

Have fun :)
